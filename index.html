Cooking Conversion

This project uses Javascript to convert various cooking units.

  1. Find the inital unit to convert.
  2. Type in how much of said unit you have to convert.
  3. Press Convert!
  4. Enjoy your converted amount.

Celebrity Dating Match

This project uses Javascript to find your mysery celebrity dating match.

  • Think about the age range and gender of your preferred celebrity match.
  • Type in you preffered age
  • Select the gender of celebrity that you're in to,
  • Click "Show me my celebrity match"
  • You're found your match. congratulations!
